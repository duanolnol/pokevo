# Pokevo

A simple Pokemon application, built with [Vite](https://vitejs.dev/), [React](https://react.dev/), [Tailwind](https://tailwindcss.com/) and [PokeAPI](https://pokeapi.co/).

## Features
- List all Pokemon
- Search Pokemon by Name
- Dark mode 
- Pokemon evolution by feeding
- Responsive design

## Setup Project

#### Prerequisites

- Install Node.js >= v18


#### Clone repo

```
git clone [repo link]
```

#### Install dependency

```
npm install
```

#### Create env file

```
Create an .env file with the following parameter: VITE_API_URL = 'https://pokeapi.co/api/v2'
```

## Usage

#### Development

```
npm run dev
```

#### Production

```
npm run build
npm run preview
```

#### Test

```
npm run test
```

#### Coverage

```
npm run cover
```
